这是一个最全面、开放且结构化的中华古典诗词文集数据库，汇聚历代诗词、经典文献及重要古籍，涵盖5.5 万首唐诗、26 万首宋诗、2.1 万首宋词及其他各种古典文集等，共计约 33.6 万首。
